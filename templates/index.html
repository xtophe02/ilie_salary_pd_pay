How it works
  1. Upload the monthly Excel payroll file (.xlsx).
  2. The app reads salary and per-diem columns automatically.
  3. IBANs from the file are saved to the bank-account database. Missing IBANs are looked up from past uploads.
  4. Two SEPA pain.001.001.03 XML files are generated — one for salaries and one for per-diem allowances.
  5. The execution date is automatically set to the next TARGET2 business day.
